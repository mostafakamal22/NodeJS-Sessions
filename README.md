## Run Locally


Clone the project


```bash

  git clone https://github.com/mostafakamal22/NodeJS-Sessions

```


Go to the project directory


```bash

  cd NodeJS-Sessions

```


Install dependencies


```bash

  npm install

```


Start the server


```bash

  npm run start

```
